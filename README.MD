# This is simple project template using React native with Expo + Native Base + Superagent

### Core Tools 
|   Package Name                |   Version     |
| ---------------               | ------------  |
|  expo                         | ~37.0.3       |
|  native-base                  | 2.13.8        |
|  superagent                   | ^5.2.2        |


# Install package
## yarn install

# Running project
## Yarn start
